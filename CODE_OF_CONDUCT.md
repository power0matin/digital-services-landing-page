# Code of Conduct

## Our Pledge

We are committed to providing a friendly, respectful, and harassment-free environment for everyone, regardless of experience level, background, identity, or perspective.

This project welcomes contributors who want to improve the codebase, documentation, design, accessibility, SEO, and overall user experience.

---

## Expected Behavior

Examples of positive behavior include:

- Being respectful and constructive
- Giving helpful feedback
- Accepting different opinions
- Focusing on what is best for the project
- Explaining suggestions clearly
- Being patient with beginners
- Keeping discussions professional

---

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment or discrimination
- Personal attacks
- Insults or hostile language
- Spam or irrelevant promotion
- Publishing private information
- Deliberate disruption of discussions
- Repeatedly ignoring maintainer guidance

---

## Project Standards

Contributors should:

- Keep discussions relevant to the project
- Be clear and concise in issues and pull requests
- Respect the maintainer's decisions
- Use issue templates where possible
- Avoid unnecessary arguments
- Provide technical reasoning for major suggestions

---

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that do not follow this Code of Conduct.

Repeated or serious violations may result in temporary or permanent restriction from participating in the project.

---

## Reporting

If you notice behavior that violates this Code of Conduct, please report it by opening a private communication channel with the maintainer.

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- Pull requests
- Discussions
- Code review comments
- Documentation contributions
- Any official project communication

---

## Attribution

This Code of Conduct is adapted for a small open-source project and is designed to keep collaboration respectful, professional, and productive.
